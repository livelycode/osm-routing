# Installation

## Erlang

### Download
1. Download Erlang/OTP R14B03 at http://www.erlang.org/download.html.
2. Unpack the download.
3. Now cd into the base dir.

### Build
    $ ./configure
    $ make

### Install
    $ make install


## Erlsom

### Download
	$ git clone https://github.com/willemdj/erlsom.git
	$ cd erlsom

### Build
	$ make

### Install
Move folder to erlang path.
